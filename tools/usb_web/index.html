M5CoreS3 USB Control

Plug in by USB, connect from Chrome or Edge, and manage Wi-Fi, receiver identity, and recovery actions without relying on network access.

Before connecting: close PlatformIO / any serial monitor using the same port (only one app can open it). Pick the usbmodem device for the CoreS3. Use a data-capable USB-C cable.

排查日志（Chrome）：按 ⌥⌘J（Mac）或 Ctrl+Shift+J（Windows）打开开发者工具，点顶部 Console。在地址栏 URL 末尾加 ?debug=1 后回车刷新，再点连接，控制台会打印 [usb_web] 开头的收发记录。
